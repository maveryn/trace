# Resource Docs

This folder contains repository-wide reusable asset guidance.
Keep procedural runbooks in `../workflows/`, repo-wide behavior contracts in
`../contracts/`, and domain-specific rendering rules in `../domains/`.

- `SHARED_LABEL_ASSETS.md` — label/name manifests, source metadata, and
  task-local filtering guidance.
- `SHARED_CONTEXT_TEXT_ASSETS.md` — non-answer context/distractor text
  manifests, source metadata, and renderer usage rules.
- `SHARED_FONT_ASSETS.md` — vendored font assets, deterministic font-family
  sampling, and text-role consistency rules.
